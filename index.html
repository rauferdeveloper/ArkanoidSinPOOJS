Instrucciones para Jugar:
Para jugar con el teclado utiliza las teclas Z o Flecha Izquierda para mover la barra hacia la izquierda y X o Flecha Derecha para mover la barra hacia la derecha. También puedes usar el raton para mover la barra pasas el cursor (puntero) donde esta el entorno de juego. Para empezar una partida usa la tecla Intro o Enter, para pausar el juego usa la tecla P y para reanudarlo utiliza la tecla R y para empezar una partida nueva dale a la tecla Espacio. Ganas el juego si destruyes todos los bloques y pierdes si la pelota pasa por debajo de la barra y toca el suelo.
